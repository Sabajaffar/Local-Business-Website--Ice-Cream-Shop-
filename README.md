## Project Overview
**HAPPY CONE** is a fully responsive static website showcasing an ice cream shop.  
The website includes:
- Home section with welcome text.
- Menu section displaying ice cream flavors, prices, and images.
- Contact section with contact details and an order form.
- About Us section describing the business.
- Responsive design for mobile, tablet, and desktop devices.
- Smooth navbar hide/show on scroll functionality.

This project is ideal for demonstrating **HTML, CSS, and basic JavaScript skills**.

---

## Technologies Used
- HTML5
- CSS3 (including media queries for responsiveness)
- JavaScript (for navbar toggle and scroll effects)
- Font Awesome (for icons)
